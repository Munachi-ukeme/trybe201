Trybe201 is a simple, static website built with HTML and CSS.  
It showcases the idea of a mentorship platform where learners can connect with expert mentors for career advice, skill development, and personal growth.

Features:
- Clean and responsive layout using only HTML & CSS  
- Sections highlighting:
- Home
- Mentorship
- Courses
- Blog

Built With
- HTML5 
- CSS3

Lessons Learned:
This project helped me practice:
- Structuring a webpage with semantic HTML  
- Styling layouts with CSS  
- Documenting projects on GitHub
